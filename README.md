## วิธีติดตั้ง

1) Run ` #Update.cmd `
2) Enter the ` index.js ` and set your prefix and bot token
3) Save ` index.js ` file
4) Run ` #RunBot.cmd `
5) Have Fun

## Install

1) รันไฟล์ ` #Update.cmd `
2) เข้าไปเเก้ไขไฟล์ ` index.js ` เพื่อเข้าไปตั่งค่า prefix เเละ Bot Token
3) เซฟไฟล์ ` index.js ` ให้เรียบร้อย
4) รัน ` #RunBot.cmd `
5) ขอให้สนุกนะค้าบ
